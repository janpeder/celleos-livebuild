Denne nettsiden kan du endre på og gjøre til din egen

Hvis du har lyst å leke deg med og øve på å lage nettsider, kan du endre på denne siden og gjøre den til din egen. Adressen du må skrive inn i nettleseren for å vise denne nettsiden er http://localhost/~elev.

For å forandre på denne nettsiden må du

  1. åpne mappa på datamaskinen som heter "public_html". Du finner denne mappa i "Hjem"-mappa på datamaskinen din.
  2. Deretter må du redigere fila "index.html" som ligger i den mappa. Du kan for eksempel redigere fila ved å bruke programmet som heter "tekstredigering".
    3. Lykke til :)
